
### 1. **`push()`**

- **Purpose**: Adds one or more elements to the end of an array and returns the new length of the array.
    

```javascript
let arr = [1, 2, 3];
arr.push(4); // arr is now [1, 2, 3, 4]
```

### 2. **`pop()`**

- **Purpose**: Removes the last element from an array and returns that element.
    

```javascript
let arr = [1, 2, 3];
let last = arr.pop(); // last is 3, arr is now [1, 2]
```

### 3. **`shift()`**

- **Purpose**: Removes the first element from an array and returns that element.
    

```javascript
let arr = [1, 2, 3];
let first = arr.shift(); // first is 1, arr is now [2, 3]
```

### 4. **`unshift()`**

- **Purpose**: Adds one or more elements to the beginning of an array and returns the new length of the array.
    

```javascript
let arr = [2, 3];
arr.unshift(1); // arr is now [1, 2, 3]
```

### 5. **`concat()`**

- **Purpose**: Merges two or more arrays into a new array without modifying the original arrays.
    

```javascript
let arr1 = [1, 2];
let arr2 = [3, 4];
let merged = arr1.concat(arr2); // merged is [1, 2, 3, 4]
```

### 6. **`join()`**

- **Purpose**: Joins all elements of an array into a single string, with an optional separator.
    

```javascript
let arr = [1, 2, 3];
let str = arr.join("-"); // str is "1-2-3"
```

### 7. **`slice()`**

- **Purpose**: Returns a shallow copy of a portion of an array (specified by start and end indices).
    

```javascript
let arr = [1, 2, 3, 4];
let sliced = arr.slice(1, 3); // sliced is [2, 3]
```

### 8. **`splice()`**

- **Purpose**: Changes the contents of an array by removing or replacing existing elements, or adding new elements in place.
    

```javascript
let arr = [1, 2, 3, 4];
arr.splice(1, 2, 'a', 'b'); // arr is now [1, 'a', 'b', 4]
```

### 9. **`forEach()`**

- **Purpose**: Executes a provided function once for each array element.
    

```javascript
let arr = [1, 2, 3];
arr.forEach(num => console.log(num)); // logs 1, 2, 3
```

### 10. **`map()`**

- **Purpose**: Creates a new array with the results of calling a provided function on every element in the array.
    

```javascript
let arr = [1, 2, 3];
let doubled = arr.map(num => num * 2); // doubled is [2, 4, 6]
```

### 11. **`filter()`**

- **Purpose**: Creates a new array with all elements that pass the test implemented by the provided function.
    

```javascript
let arr = [1, 2, 3, 4];
let even = arr.filter(num => num % 2 === 0); // even is [2, 4]
```

### 12. **`reduce()`**

- **Purpose**: Applies a function against an accumulator and each element in the array (from left to right) to reduce it to a single value.
    

```javascript
let arr = [1, 2, 3];
let sum = arr.reduce((acc, num) => acc + num, 0); // sum is 6
```

### 13. **`reduceRight()`**

- **Purpose**: Same as `reduce()`, but processes the array from right to left.
    

```javascript
let arr = [1, 2, 3];
let sum = arr.reduceRight((acc, num) => acc + num, 0); // sum is 6
```

### 14. **`some()`**

- **Purpose**: Tests whether at least one element in the array passes the test implemented by the provided function.
    

```javascript
let arr = [1, 2, 3];
let hasEven = arr.some(num => num % 2 === 0); // hasEven is true
```

### 15. **`every()`**

- **Purpose**: Tests whether all elements in the array pass the test implemented by the provided function.
    

```javascript
let arr = [1, 2, 3];
let allEven = arr.every(num => num % 2 === 0); // allEven is false
```

### 16. **`find()`**

- **Purpose**: Returns the first element in the array that satisfies the provided testing function.
    

```javascript
let arr = [1, 2, 3];
let found = arr.find(num => num > 2); // found is 3
```

### 17. **`findIndex()`**

- **Purpose**: Returns the index of the first element that satisfies the provided testing function.
    

```javascript
let arr = [1, 2, 3];
let index = arr.findIndex(num => num > 2); // index is 2
```

### 18. **`sort()`**

- **Purpose**: Sorts the elements of an array in place and returns the array. By default, it sorts them as strings.
    

```javascript
let arr = [3, 1, 2];
arr.sort(); // arr is [1, 2, 3]
```

### 19. **`reverse()`**

- **Purpose**: Reverses the order of the elements in an array in place.
    

```javascript
let arr = [1, 2, 3];
arr.reverse(); // arr is [3, 2, 1]
```

### 20. **`includes()`**

- **Purpose**: Checks if an array contains a specific element, returning `true` or `false`.
    

```javascript
let arr = [1, 2, 3];
let hasThree = arr.includes(3); // hasThree is true
```

### 21. **`indexOf()`**

- **Purpose**: Returns the first index of the element if it exists, otherwise returns -1.
    

```javascript
let arr = [1, 2, 3];
let index = arr.indexOf(2); // index is 1
```

### 22. **`copyWithin()`**

- **Purpose**: Shallow copies a portion of an array to another location in the array without modifying its length.
    

```javascript
let arr = [1, 2, 3, 4];
arr.copyWithin(0, 2); // arr is [3, 4, 3, 4]
```

### 23. **`fill()`**

- **Purpose**: Changes all elements in an array to a static value, from a start index to an end index.
    

```javascript
let arr = [1, 2, 3];
arr.fill(0, 1, 3); // arr is [1, 0, 0]
```

### 24. **`flat()`**

- **Purpose**: Flattens a nested array into a single array.
    

```javascript
let arr = [1, [2, 3], [4, 5]];
let flatArr = arr.flat(); // flatArr is [1, 2, 3, 4, 5]
```

### 25. **`flatMap()`**

- **Purpose**: First maps each element using a mapping function, then flattens the result into a new array.
    

```javascript
let arr = [1, 2, 3];
let flattened = arr.flatMap(num => [num, num * 2]); // flattened is [1, 2, 2, 4, 3, 6]
```

### 26. **`from()`**

- **Purpose**: Creates a new array instance from an array-like or iterable object.
    

```javascript
let str = "hello";
let arr = Array.from(str); // arr is ['h', 'e', 'l', 'l', 'o']
```

### 27. **`keys()`**

- **Purpose**: Returns a new Array Iterator object that contains the keys (indices) of an array.
    

```javascript
let arr = [1, 2, 3];
let keys = arr.keys(); // keys is an iterator object
```

### 28. **`values()`**

- **Purpose**: Returns a new Array Iterator object that contains the values of the array.
    

```javascript
let arr = [1, 2, 3];
let values = arr.values(); // values is an iterator object
```

### 29. **`entries()`**

- **Purpose**: Returns a new Array Iterator object that contains key/value pairs of an array.
    

```javascript
let arr = [1, 2, 3];  
let entries = arr.entries(); // entries is an iterator object

```
